## GIT AUTO PUSH 기능 설명
1. 백준 사이트에서 개인 그룹을 하나 만들어 줘야함(해당 그룹을 url로 설정해 프로그램 실행)

3. 문제를 다 풀었으면 main을 실행 -> 가끔가다가 캡챠에 걸릴때가 있는데 직접 풀어줘야한다.
